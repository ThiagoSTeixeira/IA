Esse tutorial é uma tradução/adaptação do tutorial https://www.curiousily.com/posts/build-your-first-neural-network-with-pytorch/

O texto do tutorial encontra-se no arquivo Tuturial.pdf, enquanto código-fonte se encontra no diretório src/

Os dados para o treinamento podem ser encontrados em https://www.kaggle.com/jsphyg/weather-dataset-rattle-package
Baixe e extraia o arquivo de dados dentro do diretório src/data/

É sugerido que você leia o pdf antes de explorar o código.

Para executar todo o fluxo de pré-processamento dos dados, de treinamento e uso da rede, faça:

python3 main.py

Para rodar os testes automáticos da implementação, faça:

python3 tests.py
